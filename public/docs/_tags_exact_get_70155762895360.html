GET /tags/exact

Description

Getting a tag by exact name

Headers

Accept

application/vnd.hairfolio.v1

Content-Type

application/json

Parameters

q required, type: string

The tag name

Example Response

{
    "tag": {
        "id": 360,
        "created_at": "2016-12-11T18:14:29.339Z",
        "name": "voluptatem",
        "last_photo": "https://jkfslf.com/image.png",
        "url": null
    }
}

Resource Information

API Version 1.0
HTTP Method GET
Login Required false
Response Format application/json
