POST /salons

Description

Creating a salon

Headers

Accept

application/vnd.hairfolio.v1

Content-Type

application/json

Parameters

salon[name] required, type: string

The name of the salon

salon[info] required, type: string

The info of the salon

salon[address] required, type: string

The address of the salon

salon[city] required, type: string

The city of the salon

salon[state] required, type: string

The state of the salon

salon[zip] required, type: string

The zip of the salon

salon[phone] required, type: string

The phone of the salon

salon[website] required, type: string

The website of the salon
