PUT /conversations/:conversation_id/messages/:id

Description

Updating a message in a conversation

Headers

Accept

application/vnd.hairfolio.v1

Content-Type

application/json

Parameters

message[body] required, type: string

THe body of the message

message[url] required, type: string

THe url link of the message

message[post_id] required, type: integer

A post to attach to the message
