PUT /posts/:id

Description

Updating a post

Headers

Accept

application/vnd.hairfolio.v1

Content-Type

application/json

Parameters

post[description] required, type: string

This is the body of the post

post[photos_attributes][0][asset_url] required, type: string

This is the service of the formula to add

post[photos_attributes][0][id] required, type: string

This is the id of the photo to update

post[photos_attributes][0][_destroy] required, type: string

Set the destroy flag to delete teh photo

post[photos_attributes][0][labels_attributes][0][id] required, type: integer

This is the id of the label to update

post[photos_attributes][0][labels_attributes][0][_destroy] required, type: integer

Set the _destroy flag to remove the label

post[photos_attributes][0][labels_attributes][0][tag_id] required, type: integer

This is the id of the tag to associate the post with

post[photos_attributes][0][labels_attributes][0][position_top] required, type: integer

This is the position from the top to place the tag

post[photos_attributes][0][labels_attributes][0][position_left] required, type: integer

This is the position from the left to place the tag

post[photos_attributes][0][labels_attributes][0][formulas_attributes][0][id] required, type: integer

This is id of the formula to update

post[photos_attributes][0][labels_attributes][0][formulas_attributes][0][_destroy] required, type: integer

Set the _destroy flag to delete teh formula from the photo

post[photos_attributes][0][labels_attributes][0][formulas_attributes][0][service_id] required, type: integer

This is the service of the formula to add

post[photos_attributes][0][labels_attributes][0][formulas_attributes][0][weight] required, type: integer

This is the weight of the formula to add

post[photos_attributes][0][labels_attributes][0][formulas_attributes][0][volume] required, type: integer

This is the volume of the formula to add

post[photos_attributes][0][labels_attributes][0][formulas_attributes][0][time] required, type: integer

This is the time of the formula to add

post[photos_attributes][0][labels_attributes][0][formulas_attributes][0][line_id] required, type: integer

This is the id of the color line

post[photos_attributes][0][labels_attributes][0][formulas_attributes][0][treatments_attributes][id] required, type: integer

This is the id of the formula treatment

post[photos_attributes][0][labels_attributes][0][formulas_attributes][0][treatments_attributes][_destroy] required, type: integer

Set the _destroy flag to delete the treatment from the formula

post[photos_attributes][0][labels_attributes][0][photos_attributes][0][formulas_attributes][0][treatments_attributes][weight] required, type: integer

This is the weight of the formula treatment

post[photos_attributes][0][labels_attributes][0][formulas_attributes][0][treatments_attributes][color_id] required, type: integer

This is the color id of the formula treatment
