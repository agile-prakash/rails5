GET /posts

Description

Getting the post feed for the logged in user

Headers

Accept

application/vnd.hairfolio.v1

Content-Type

application/json

Parameters

popular optional, type: boolean

Setting the popuar toggle filters by popular

favorites optional, type: boolean

Setting the favorites toggle filters by popular
