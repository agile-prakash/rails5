PUT /contacts/:id

Description

Updating a contact

Parameters

contact[first_name] required, type: string

The first name of the contact

contact[last_name] required, type: string

The last name of the contact

contact[address] required, type: string

The address of the contact

contact[city] required, type: string

The city of the contact

contact[state] required, type: string

The state of the contact

contact[zipcode] required, type: string

The zipcode of the contact

contact[company] required, type: string

The company name of the contact

contact[emails_atributes][0][id] required, type: string

The id of the email to be updated.

contact[emails_atributes][0][email] required, type: string

The email address of the email

contact[emails_atributes][0][email_type] required, type: string

The email type of the email. (work or home)

contact[emails_atributes][0][_destroy] required, type: string

Set the _destroy flag to delete the email from the contact

contact[phones_attributes][0][id] required, type: string

The id of the phone

contact[phones_attributes][0][number] required, type: string

The number of the phone

contact[phones_attributes][0][phone_type] required, type: string

The type of the phone contact (home or work)

contact[phones_attributes][0][_destroy] required, type: string

Set the _destroy flag to delete the phone from the contact

contact[post_ids] optional, type: array

An array of post ids
