POST /users/:user_id/offerings

Description

Creating a user offerings

Headers

Accept

application/vnd.hairfolio.v1

Content-Type

application/json

Parameters

offering[price] optional, type: integer

The price of the offering

offering[service_id] optional, type: integer

The id of the service

offering[category_id] optional, type: integer

The id of the category
