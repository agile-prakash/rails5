PUT /photos/:id

Description

Updating a photo

Headers

Accept

application/vnd.hairfolio.v1

Content-Type

application/json

Parameters

photo[asset_url] optional, type: string

The url of the photo

photo[post_id] optional, type: integer

The id of the post

photo[video_url] optional, type: string

The url of the video
