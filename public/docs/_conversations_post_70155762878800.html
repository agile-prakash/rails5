POST /conversations

Description

Starting a conversation with users

Parameters

conversation[recipient_ids] required, type: array

Array of user ids in the conversation

Example Response

{
    "conversation": {
        "id": 160,
        "sender_id": 2331,
        "created_at": "2016-12-10T00:38:45.377Z",
        "last_message": null,
        "unread_messages": 0,
        "recipient_ids": [
            2333
        ]
    }
}

Resource Information

API Version 1.0
HTTP Method POST
Login Required true
Response Format application/json
